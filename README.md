### Assignment

* Prepare DB information: connection information, schema name,..
* Create small Java program using JDBC to connect DB, demontrate following functions:
  * Connect and close connection
  * Create statement to get all schema, table entry, column name, attribute and constraint...
  * Data manipulation: query all necessary information and build entity model includes properties.
    * Define `TableModel.class`
    * Define `ColumnModel.class`
* How to build `EntityName`: built from `TableModel.name`, split by `_`, uppercase first character of each word. If last word is `M`, it will become `Mirror`. If last word is `UD`, it will become `UD`. In general, the `EntityName` follows CamelCase.
* Supports to export entity model to variety format: Excel, CSV, Json.

### Spring Assignment

* Artifact for this training is `spring-training.zip`.
* Trainee needs to extract and build using `Maven`.
* Trainee downloads [`ojdbc7.jar`](http://www.oracle.com/technetwork/database/features/jdbc/jdbc-drivers-12c-download-1958347.html)
* Install to Maven local repository by `mvn install:install-file -Dfile=<PATH-TO-OJDBC7.jar> -DgroupId=com.oracle -DartifactId=ojdbc7 -Dversion=12.1.0.1.0 -Dpackaging=jar`
* Configure to access to Oracle database (training schema)
* Implement CRUD service for `COMPANY` object
  * Tables in scope is `FBCOMPANY`, `FBCOMPANY_UD`, `FBUSERGRP_COMPANY`, `FBUSERGRP`, `FBUSER`
  * Search list of `COMPANY`: to return list of `CompanyDTO` object which `DELETE_RESERVED` = 0. Search list will not return full information of `COMPANY` object. REST API */company_list*.
  * Search detail information of `COMPANY`: search by companyID and return full content of `COMPANY`. REST API */company_detail*.
  * Update the `COMPANY` information: receive a `CompanyDTO` with full relationship and update to database. REST API */company_update*.
  * Create the `COMPANY` information: receive a `CompanyDTO`, validate if this companyId is not exist in database, then create new one to database. REST API */company_create*.
  * Delete the `COMPANY`: modify `DELETE_RESERVED` to 1. REST API */company_delete*.
